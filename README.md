# Calabiyau 表情包项目

适用于 OwO 表情包标准格式的 卡拉彼丘 表情包集合。

## 项目简介

本项目包含 192 个 卡拉彼丘 表情包，采用 OwO 表情包标准格式，适用于Artalk、Twikoo等评论系统。

## 使用方法

### 1. OwO 格式

本项目遵循 OwO 表情包标准格式，配置文件如下：

```json
{
  "name": "卡丘",
  "type": "image",
  "items": [
    {
      "key": "1",
      "val": "https://cdn.jsdelivr.net/gh/Elvish064/Calabiyau-stickers/calabiyau/1.gif"
    },
    // ... 其他190个表情包
    {
      "key": "192",
      "val": "https://cdn.jsdelivr.net/gh/Elvish064/Calabiyau-stickers/calabiyau/192.gif"
    }
  ]
}
```

### 2. 在Artalk、Twikoo等评论系统中使用

 在配置文件中加入`calabiyau.json`.
```

// existing code...
 },
  "https://cdn.jsdelivr.net/gh/qwqcode/huaji/huaji.json",
  "https://cdn.jsdelivr.net/gh/Elvish064/Calabiyau-stickers/calabiyau.json" //在此处插入
]

```

#### 或者使用defalut.json(自用)替换原有配置文件

修改Artalk表情包配置文件为：`https://cdn.jsdelivr.net/gh/Elvish064/Calabiyau-stickers/default.json`

其中bilibili表情包来自：[Twikoo-Magic](https://github.com/noionion/Twikoo-Magic)

- **格式**: GIF
- **数量**: 192 个
- **分辨率**: 300x300
- **文件命名**: 1.gif 至 192.gif
- **CDN**: jsDelivr
